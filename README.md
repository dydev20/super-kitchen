# Super Kitchen 

## Description
*

#### Features 
* Add food items to order from menu 
* Users can select, where applicable: 
  * The style the food item is cooked in 
  * Additional options for the food item
  * Side item for meals 
* View order summary
* Remove items from order
* Calculate total price for the order

#### Technologies
* React
* Tailwind CSS
